find attached the excel spreadsheets related to excess mortality.
EM_Data_availability: metadata file
COVID_Excess_Mortality_v211207: the final excel file of the absolute number of excess mortality in each regions (Excess Mortality: It is calculated by measuring the number of deaths from all causes in a given period with the average number of deaths in the same period in the last 5 years (2015-2019). Excess Mortality data are available on a weekly basis and therefore fit well with other epidemiological data which were also collected in weekly.
Excess_mortality_database_220103: the original dataset; weekly deaths 2015-2021, furthermore the absolute and relative (per 10.000 inhabitants) number of excess mortality (and some visualisations)
